# 🗳️ Online Voting System – Web Application

## 📖 Overview
The **Online Voting System** is a secure and user-friendly web application designed to conduct elections electronically.  
It enables **voters to cast their votes online** with authentication, ensuring transparency, accessibility, and efficiency.  
The system eliminates the need for manual voting and counting, providing instant results and improved election integrity.

---

## ⚙️ Features
- **User Registration & Login:** Secure voter and admin authentication.  
- **Admin Dashboard:** Manage voters, candidates, and election details.  
- **Voting Module:** Each voter can cast one vote per election securely.  
- **Result Display:** Instant display of total votes and winners.  
- **Database Integration:** Uses MySQL for storing and managing election data.  
- **Responsive Design:** Works smoothly on desktops, tablets, and mobile devices.

## 🚀 Installation and Setup
Follow these steps to set up and run the project locally:

1. **Install Requirements**
   - [Download XAMPP](https://www.apachefriends.org/download.html) or WAMP server.  
   - Start **Apache** and **MySQL** services.

2. **Setup Database**
   - Open **phpMyAdmin**.  
   - Create a new database named `online_voting_system`.  
   - Import the provided `.sql` file from the project folder.

3. **Configure the Project**
   - Extract the project folder into `htdocs` (for XAMPP) or `www` (for WAMP).  
   - Update database credentials in `config.php` (if required).  

4. **Run the Application**
   - Open browser and go to:  
     ```
     http://localhost/OnlineVotingSystem/
     ```
   - Login using admin credentials or register as a new voter.
